## Overview

This project is a personal hard-surface modeling study focused on creating a
sci-fi industrial robotic arm (manipulator).

The goal is to practice:
- mechanical design logic
- clean hard-surface topology
- boolean-based workflows
- readable sci-fi industrial aesthetics

This repository documents the **process and progression** of the project.

## Project Focus

- Hard-surface modeling
- Mechanical joints and pivots
- Industrial sci-fi design
- Clean topology after booleans
- Non-destructive workflow where possible
