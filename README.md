# English

## Overview

This project is a hard-surface robotic arm study made during the
BlenderBros Hard Surface Modeling Jumpstart course.

The project focuses on:
- mechanical design logic
- clean hard-surface topology
- boolean-based workflows
- readable sci-fi industrial aesthetics

This repository documents the **process and progression** of the project.

## Project Focus

- Hard-surface modeling
- Mechanical joints and pivots
- Industrial sci-fi design
- Clean topology after booleans
- Non-destructive workflow where possible

## Structure

```
├── SCI-FI_Robotic_arm.blend
├── progress/
│ ├── blockout.png
│ ├── mid.png
│ └── final.png
├── renders/
│ └── final_render.png
└── README.md
```

**SCI-FI_Robotic_arm.blend** — main Blender project file.  
**progress/** — WIP screenshots showing key stages of the modeling process.  
**renders/** — final renders of the project.

## Progress Tracking

Progress is tracked via:
- Git commit history
- WIP screenshots in `progress/`

Each commit represents a **logically finished step**
(e.g. blockout → joints → paneling → cleanup → render).
